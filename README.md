# requirements
Some features rely on bash version 4 functions not available earlier.

# usage
Clone repo to ~/.bash/autobash/ and add to .bash_profile:

```
export AUTOBASH=~/.bash/autobash
source "${AUTOBASH}/main.sh"
```
